==> 在 rstsai.bbs@csie.nctu (HMTechica) 的文章中提到:
: 前言
: 在以往通訊器材管制的時代，山難搜救必定要發動軍警配合。現在無線電手機以至於
: 俗稱大哥大的行動電話均已相當普遍，登山團體實際上已有能力自己聯合進行搜救工
: 作。但因組織不健全和觀念不普及，現在山難搜救大多仍由一些對山區狀況不瞭解的
: 警察及山青所主導，因此在效率及物資人力的調配上，仍不盡理想。這篇文章的目的
: ，就是希望能刺激各位多思考些關於山難搜救的問題，以建立一個完善的互助體制。
: 第一要務____通訊線的建立
: 以台灣現在的通訊資源，在山區活動的人員很難與外界保持聯繫。遇有山難狀況發生
: 時，外界支援的人員首先要設法瞭解隊伍的狀況，亦即要建立一條能和出事隊伍通訊
: 的管道。其次要針對其需要，派遣人員並運送必要物資進入山區予以援助，為控制此
: 一工作能順利達成，支援搜救的人員亦應與其指揮中心保持聯繫。這兩個山難搜救的
: 主要步驟，其實都依賴著通訊網路的建立來達成。換言之，通訊線的建立其實是山難
: 搜救的第一個步驟。
: 架設無線電通訊網路其實並不是很困難的事，如果無法使用自動轉接，則要派駐人員
: 專司通訊工作。這裡只特別提出幾點容易被忽略的細節：
: 1.各社團在平時即應舉行手機使用的演練，並要求隊伍養成帶手機與收音機的習慣。
: 2.隊伍在出發前應與留守人員約定好遲歸後的通話時間和頻道，以備不時之需。
: 3.出事的隊伍若已無法推進，則應儘量紮營在稜線等易於通訊之處，並設法與外界連
:   絡。在近年發生的一些案例中，多半是先與當地的火腿（或香腸）族取得聯繫，再
:   輾轉通知留守人及警方。
: 4.在選擇專司通訊的人員之駐守位置時，除考慮其生活上之條件（水、營地、安全）
:   外，亦應注意電波是否會受到阻礙。
: 5.任何搜救行動必需以活著的人的安全為首要考量。派駐的通訊人員必須要有確保其
:   自身安全的能力，若遇惡劣天候或其他必要狀況，應及早撤出。
: 6.通訊人員應帶紙筆或其他記錄工具，將所有通話記錄下來，以供事後檢討。

   rstsai兄:  好久不見!
 
   我這條命是靠火腿族及對講機.直昇機和海鷗部隊救出來的
   再加上近年山社出了幾次山難
   關於通訊器材在山難事件的使用過程稍稍說明
 
   山難現場的對講機使用者請切記要求接收的火腿族
   覆述傳話內容確定無誤才可傳出去
   ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
   每一次的的轉話必需有覆述確認的過程
   現場使用者記得要用紙筆記下通訊的時間及內容
   掌握何時?何地?何人?何事?求援事項等要點精簡呼救
 
   80年的素密達事件報上由火腿族誤傳寫出了
   "大霸與品田間的森林燃燒殆盡,山青辛苦下溪谷背水搶救"
   這些誇大編派的情節(大火是靠當時在場的師大.成功山社
   男山胞以尿液澆息,山青未達現場火勢即控制)
   然而出院後兩個月我重回現場,只看見半棵燒焦的樹幹
   學弟起火引導直昇機救人還被林務局告發,跑了幾趟法院
   罰了1499元(一棵樹的賠償費)
 
   今年暑假本社能安隊,帶了一支只能發話而不能接收的對講機上山
   正值颱風蘆山地區有人受傷
   火腿族誤傳清大山社有人失蹤
   當時公路由埔里到霧社柔腸寸段
   社上在四小時之內聚集二十多人
   完成採買預定發動搜救隊
   結果連絡官馬不停蹄趕到現場才知
   這只是單純的遲歸事件           
  
   今年五月的玉山失蹤搜救
   參與搜救的民間團體將對講機鎖定
   社上搜救隊無法解碼也造成了不少困擾
 
   一般而言,社會上的搜救對伍多半是火腿族而非登山族
   就搜救能力而言自是比不上學校登山團體
   然而學校登山社的對講機使用能力缺乏實務經驗及專門課程的學習
   無法達到"工欲善其事,必先利其器"的效果
    
   通訊器材的機種日新月異,操作過程也不盡相同
   如何克服使用技術的缺憾是我們應該努力思考改進的方向
